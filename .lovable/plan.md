
# AI-Generated Images for Blog Posts with Diversity and SEO Alt Tags

## Overview

Switching from stock photos to AI-generated images for all blog images (featured and middle), with Pixabay as a fallback. Each image will be unique, realistic, diverse in style/color, and include SEO-optimized alt text.

---

## Key Changes

### 1. AI Image Generation for All Images
- **Featured Image**: Generated by AI based on the article title and category
- **Middle Images (1-2)**: Generated by AI with varied prompts to ensure diversity
- **Fallback**: If AI generation fails, use Pixabay as backup

### 2. Sequential Processing
- Process articles one at a time (not in parallel)
- Wait for all images to complete before starting the next article
- Increased delay between articles (5-8 seconds) to avoid rate limiting

### 3. Image Diversity
Each image prompt will include:
- Randomized style variations (warm tones, cool tones, high contrast, soft light, etc.)
- Different scene compositions per image within the same article
- Explicit instructions to avoid repetitive color schemes

### 4. SEO-Optimized Alt Tags
- Add database columns for alt text storage
- Generate descriptive alt text using AI based on article topic
- Display alt text in ArticlePage rendering

### Style Rotation for Diversity

```typescript
// Rotate through styles to ensure variety across images
const STYLE_VARIANTS = ['warm', 'cool', 'neutral', 'dramatic'] as const;

// For each article, randomly assign different styles
function getImageStyles(): { featured: string; middle1: string; middle2: string } {
  const shuffled = [...STYLE_VARIANTS].sort(() => Math.random() - 0.5);
  return {
    featured: shuffled[0],
    middle1: shuffled[1],
    middle2: shuffled[2],
  };
}
```

## Expected Behavior

1. **Generation Time**: Longer per article (AI image generation takes 10-20 seconds each)
2. **Sequential Processing**: One article completes fully before the next starts
3. **No Duplicates**: Each AI-generated image is unique
4. **Visual Diversity**: Randomized styles prevent uniform appearance
5. **SEO Friendly**: All images have descriptive, keyword-rich alt tags
6. **Resilient**: Pixabay fallback prevents failures from breaking generation
